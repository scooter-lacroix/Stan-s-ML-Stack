# claude-hud Configuration for Maestro

This project uses claude-hud for native token tracking.

**Installation:** Pending (manual install required)
**Statusline Configured:** Yes
**Configuration Date:** 2026-02-17

## Installation Instructions

Install claude-hud using one of these methods:

```bash
# Via cargo
cargo install claude-hud

# Or check the official repository
# https://github.com/Cline-org/claude-hud
```

## Statusline Features

Once installed, claude-hud will show:
- Current Maestro command
- Track/task context
- Real-time token usage
- Cost estimates

## Notes

claude-hud provides native tracking without separate API calls.

See https://github.com/Cline-org/claude-hud for more information.
